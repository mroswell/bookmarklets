This bookmarklet displays a bl.ocks page from a GitHub Gist.
  1. Install the bookmarklet by dragging this link to your browser's bookmark toolbar. (Ensure that the toolbar is showing first.)
    bl.ocks
  2. Visit a GitHub Gist that includes an index.html file. For instance: Mike Bostock's Colorcomb
  3. Press your new 'bl.ocks' button to go to the associated bl.ocks.org page.
Notes:
  • In some cases on the bl.ocks.org page, you'll need to click the "Open in a new window" link to see the visualization.
  • You can use bookmarklets in Chrome on Android and iOS.
  • To see a similar bookmarklet, along with a video of how to use it, visit MapVote
